# Ask a question about your codebase
```bash
admino ask "How do I implement rate limiting?"
```

```bash
admino generate-test open_code/api.py
```

```bash
admino interactive
```

```bash
admino edit open_code/main.py "Add better error handling"
```
# View your configuration
```bash
admino config
```

```bash
admino history
```
